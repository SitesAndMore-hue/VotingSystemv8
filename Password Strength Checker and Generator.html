How to Create Strong Passwords

Do's of Password Security

  • Use long passwords

    Aim for at least 12-16 characters. Length is more important than complexity.

  • Use a mix of character types

    Combine uppercase, lowercase, numbers, and symbols.

  • Use passphrases

    Combine multiple random words: "CorrectHorseBatteryStaple".

  • Use a password manager

    Generate and store unique passwords for every account.

  • Enable two-factor authentication

    Add an extra layer of security beyond passwords.

Don'ts of Password Security

  • Don't use personal information

    Avoid names, birthdays, pet names, or addresses.

  • Don't reuse passwords

    Use unique passwords for every account.

  • Don't use common patterns

    Avoid "123456", "qwerty", "password", or sequential patterns.

  • Don't write passwords down

    If you must, keep them in a secure, encrypted location.

  • Don't share passwords

    Never share passwords via email, text, or messaging apps.

How Hackers Crack Passwords

Dictionary Attacks

Hackers use lists of common words, phrases, and previously leaked passwords. If your password is in a dictionary, it can be cracked instantly.

Brute Force Attacks

Computers try every possible combination of characters. Short passwords with limited character sets are vulnerable to this method.

Social Engineering

Hackers gather personal information from social media to guess passwords based on your life: pets, birthdays, hobbies, etc.
